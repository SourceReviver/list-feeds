Lists all available feeds (rss,atom) of a page. RSS and Atom feeds are still a very useful method to keep up to date on news and other kind of information. On many sites the feed capabilites are still available but sometimes badly maintained, so that it can to hard to access or find these. This Add-On Tries to make it easy to find all available feeds a site has to offer. Usage:
  1. Visit a site which contains feeds for example:
    • https://en.wikipedia.org/wiki/RSS
    • https://www.youtube.com/user/Mozilla
    • https://www.youtube.com/channel/UC-CBSBRP82j2t5PQ3M33CVw
    • https://www.youtube.com/watch?v=4fviHC7xlX0&list=PLnRGhgZaGeBvDBzQGf4M8HIepPXA9cq-C
  2. Click the list-feeds pageAction icon in the url bar.
  3. Select or Open the feeds in the the popup window below the pageAction icon.
